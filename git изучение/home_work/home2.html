Добавление файлов
Создайте проект с тремя файлами: index.html, index.js, index.css.
Выполните следующие команды:
добавьте в стадию ожидания лишь файл с расширением .js;
выполните проверку статуса всего проекта;
добавьте также файл с название index.css;
выполните коммит;
проверьте статус проекта.

Игнорирование
Создайте файл для игнорирования данных в проекте.
Внутри этого файла укажите игнорирование таких объектов, как:
все файлы с расширением .txt;
все файлы внутри папки: admin, bin, config;
все файлы в папке user, кроме файла auth.php.
